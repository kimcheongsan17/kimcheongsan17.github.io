# Cheongsan Kim Academic Portfolio

Personal GitHub Pages website for Cheongsan Kim.

## Structure

- `index.html`: main portfolio with tabbed navigation
- `styles.css`: responsive layout and visual design
- `script.js`: tab switching behavior
- `papers/`: detail pages for selected papers
- `projects/`: detail pages for selected research projects
- `assets/profile.jpg`: optional profile photo path

To add a profile photo, place an image at `assets/profile.jpg`.
